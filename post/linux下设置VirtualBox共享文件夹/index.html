linux下设置VirtualBox共享文件夹

By 中正

发表于 2013-02-06

文章目录

设置个VirtualBox共享文件夹设置了三四个下午，终于设置完了，真是内牛满面～～

先介绍下环境：

HOST： Ubuntu 12.10   

GUEST: Win XP 

VirtualBox: 4.2.6 （可以用命令 VBoxManage -version 查看版本）

PS: VirtualBox官网：[https://www.virtualbox.org/][https_www.virtualbox.org]

       安装VirtualBox最好到官网下载后再安装，因为官网的版本是最新的，有相应的增强补丁；用ubuntu software center 搜索安装就比较坑了。

第一步：Guest上安装Guest Additions

如果你RP够好的话，

运行虚拟机，在Tool Bar上找到 Device下拉菜单，选择 Add Guest Additions，然后在虚拟机中点“下一步....“即可安装完毕。

如果你和我一样，悲剧的发现找不到Tool Bar，

先对虚拟机 Right Ctrl + C切换到 Scale mode,然后再 Right Ctrl + HOME 就会出现梦寐以求的 Tool Bar!!!

然后重复刚刚的操作。

重启虚拟机.

第二步：HOST上设置Linux共享文件夹

打开shell, 输入 “VBoxManage sharedfolder add “test 1” –name “tmp” –hostpath “/tmp””

//这句命令的意思是， 把 HOST 的 “/tmp” 文件夹， 取名为 “tmp”, 共享给 “test 1”虚拟机。

第三步: GUEST上映射文件夹

打开cmd， 输入 “net use x: \\vboxsvr\tmp”

//这句话的意思是, 把 HOST 上的 tmp 共享文件夹 映射为 本地的 X 盘。

至此，大工告成了！！
